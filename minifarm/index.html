Getting Started
Setup Guide
01

Install MiniFarm

Download the MiniFarm .dmg installer and drag MiniFarm.app to your Applications folder. Launch it — it will appear in your menu bar.

02

Set the aerender Path

Open MiniFarm → Settings and click Browse next to "aerender Path". Navigate to your After Effects installation and select aerender.

/Applications/Adobe After Effects [version]/aerender
💡 The version folder name varies — e.g. "Adobe After Effects 2024".
03

Set Working & Output Directories

In Settings, set your Working Directory to a shared folder accessible by all nodes on your network (e.g. a NAS or shared volume). Set your Output Directory where rendered frames will be saved.

/Volumes/SharedStorage/MiniFarm/Work
/Volumes/SharedStorage/MiniFarm/Output
04

Create the "minifarm" Output Module in After Effects

Open After Effects → Edit → Templates → Output Module. Click New, name it exactly minifarm (lowercase), configure your desired codec and format, then click OK and save.

⚠️ The output module name must be exactly minifarm for MiniFarm to recognise it.
05

Install the After Effects Plugin

Copy the MiniFarm.jsx script to your After Effects Scripts UI Panel folder, then enable it via File → Scripts → MiniFarm. Use the plugin panel inside AE to send jobs to the desktop app.

/Applications/Adobe After Effects [version]/Scripts/ScriptUI Panels/
01

Install MiniFarm

Download the MiniFarm installer (.exe) and run it. MiniFarm will install and appear in your system tray after launch.

02

Set the aerender Path

Open MiniFarm → Settings and click Browse next to "aerender Path". Navigate to your After Effects installation and select aerender.exe.

C:\Program Files\Adobe\Adobe After Effects [version]\Support Files\aerender.exe
💡 Replace [version] with your installed version, e.g. "2024".
03

Set Working & Output Directories

In Settings, point Working Directory to a network share accessible by all render nodes. Set your Output Directory for rendered frames.

\\NAS\MiniFarm\Work
\\NAS\MiniFarm\Output
⚠️ Use UNC paths (\\server\share) rather than mapped drive letters for cross-node compatibility.
04

Create the "minifarm" Output Module in After Effects

Open After Effects → Edit → Templates → Output Module. Click New, name it exactly minifarm (lowercase), configure your desired codec and format, then click OK and save.

⚠️ The output module name must be exactly minifarm for MiniFarm to recognise it.
05

Install the After Effects Plugin

Copy MiniFarm.jsx to your After Effects ScriptUI Panels folder, then enable via File → Scripts → MiniFarm. Use the panel to send jobs directly to MiniFarm.

C:\Program Files\Adobe\Adobe After Effects [version]\Support Files\Scripts\ScriptUI Panels\
